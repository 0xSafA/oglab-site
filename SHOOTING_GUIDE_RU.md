## 🎥 Hero Video (20-30 секунд)

### Что показывать:

```
SHOT LIST:

1. [0:00-0:05] - Восход над плантацией
   📸 Wide shot, панорама фермы

2. [0:05-0:10] - Крупный план растений
   📸 Close-up, зеленые листья на ветру

3. [0:10-0:15] - Руки ухаживают
   📸 Medium shot, руки обрезают/поливают

4. [0:15-0:20] - Витрина
   📸 Medium shot, продукты на полках

5. [0:20-0:25] - Улыбающийся персонал
   📸 Close-up, лицо сотрудника с улыбкой

6. [0:25-0:30] - Логотип
   📸 Close-up, логотип OG Lab
```

### Технические требования:

```
✅ Разрешение: 1920x1080 (Full HD) минимум
✅ FPS: 24-30 fps
✅ Формат: MP4 (H.264)
✅ Битрейт: 800-1500 kbps (после сжатия)
✅ Размер: < 10 MB (обязательно!)
✅ Звук: можно без звука (muted)
```

### Советы по съемке:

```
✅ ХОРОШО:
  • Штатив или стабилизатор
  • Golden hour (за час до заката)
  • Плавные движения камеры
  • Чистые линзы
  • Хорошее освещение

❌ ПЛОХО:
  • Трясущаяся камера
  • Полдень (жесткий свет)
  • Резкие движения
  • Грязные линзы
  • Тени на объектах
```

## 😊 Фото команды (4 человека)

### Кого снимать:

```
1. 👨‍🌾 АГРОНОМ (Head Cultivator)
   - На фоне растений
   - Можно с инструментами
   - В теплице или на плантации

2. 👩‍💼 МЕНЕДЖЕР (Customer Experience)
   - В диспенсари
   - Можно за кассой
   - Или на входе

3. 👨‍🔬 КОНТРОЛЬ КАЧЕСТВА (Quality Control)
   - В лаборатории
   - Можно с микроскопом/оборудованием
   - Или с образцами

4. 👩‍🏫 ГИД (Farm Tour Guide)
   - На плантации
   - Можно показывает растения
   - Или с группой туристов
```

### Технические требования:

```
✅ Формат: Квадратный (1:1) ОБЯЗАТЕЛЬНО!
✅ Размер: 800x800 минимум
✅ Файл: < 300 KB
✅ Формат: JPG
✅ Фон: Размытый (bokeh) или ферма
✅ Композиция: Лицо занимает 60-70% кадра
```

### ⚠️ САМОЕ ВАЖНОЕ:

```
✨ УЛЫБКА! УЛЫБКА! УЛЫБКА! ✨

✅ Искренняя улыбка (не напряженная)
✅ Зубы видны (не застенчивая)
✅ Глаза "улыбаются" тоже
✅ Естественная поза
✅ Расслабленные плечи

❌ Серьезное лицо
❌ Натянутая улыбка
❌ Напряженная поза
❌ Отведенный взгляд
```

### Примеры хороших фото:

### Поза и композиция:

```
КОМПОЗИЦИЯ:
┌────────────────────────────┐
│        [НЕБО/ФОН]          │
│                            │
│      ┌──────────┐          │
│      │   😊    │          │ ← Лицо в верхней трети
│      │  /│\    │          │
│      │   👔    │          │

Плечи: под углом 15-30° (не прямо)
Руки: естественно (не скрещены)
```

### Настройки камеры:

```
📷 Рекомендуемые настройки:
  • Aperture: f/2.8 - f/4 (размытый фон)
  • ISO: 100-400 (чистая картинка)
  • Shutter: 1/125+ (резкость)
  • Focal length: 50-85mm (портрет)
  • White balance: Daylight/Auto
```
